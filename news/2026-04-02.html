🛡️ Intel Briefing: April 02, 2026

Daily Cyber Intelligence Brief Date: April 1, 2026 To: Technical Leadership and Security Operations From: Senior Security Research Team --- ### 1. MAIN HEADLINE North Korean State Actors Compromise Axios npm Package in Massive Supply Chain Attack ### 2. THE BIG STORY: Sapphire Sleet Poisoning of Axios Ecosystem Microsoft Threat Intelligence has confirmed a major supply chain compromise affecting Axios, the ubiquitous JavaScript HTTP client. Attributed to the North Korean state actor Sapphire Sleet, the attack involved the publication of malicious updates—versions 1.14.1 and 0.30.4—to the npm registry. These tainted packages were configured to automatically communicate with command-and-control (C2) infrastructure to facilitate secondary payloads. The vulnerability stems from an unauthorized takeover of the publishing pipeline, allowing the actor to inject scripts into the package lifecycle. Because Axios is a foundational dependency for millions of web applications and CI/CD pipelines, the blast radius is exceptionally wide. While the malicious versions have been removed from the npm registry, any build systems or developers who pulled these specific versions between March 31 and April 1 must assume full environment compromise. The technical mechanism utilizes a standard dependency confusion or account takeover tactic to deliver the C2-linked code. Once the package is initialized in a Node.js environment, it executes a post-install or runtime hook that beacons to Sapphire Sleet's infrastructure. This provides the threat actor with a foothold in developer workstations and production servers, enabling the exfiltration of environment variables, source code, and cloud credentials. ### 3. VENDOR SECURITY WATCH * Microsoft: Issued urgent mitigation guidance for the Axios npm compromise; also warned of a new VBS malware campaign delivered via WhatsApp that utilizes a UAC bypass to establish persistent remote access on Windows systems. * Cisco (Talos): Disclosed and coordinated patches for 30 vulnerabilities: one in HikVision (Remote Code Execution), 10 in TP-Link networking gear, and 19 in Canva's infrastructure. * Cloudflare: Released "Programmable Flow Protection" for Magic Transit, allowing engineers to deploy custom, stateful L3/L4 mitigation logic for proprietary UDP-based protocols. * Fortinet: *No critical updates reported in this cycle.* * Oracle: *No critical updates reported in this cycle.* ### 4. CRITICAL HEADLINES * TrueConf Zero-Day Exploitation: Threat actors are actively exploiting a zero-day vulnerability in TrueConf video conferencing servers to push malicious software updates and execute arbitrary files on all connected endpoints. * 'CanisterWorm' Wiper Hits Iran: A new destructive worm is targeting Iranian infrastructure. The malware checks for Farsi language settings or Iranian time zones before initiating a data-wiping routine; it spreads via misconfigured cloud services. * Global IoT Botnet Takedown: The DOJ, in coordination with German and Canadian authorities, dismantled the infrastructure for the Aisuru, Kimwolf, JackSkid, and Mossad botnets, which controlled over three million compromised IoT devices. * Apple Expands iOS 18 Protections: Apple has backported security updates to older iPhone models to mitigate the DarkSword exploit kit, which is currently being used in active, targeted attacks. * CERT-UA Impersonated: A phishing campaign by UAC-0255 is targeting Ukrainian entities with AGEWHEEZE malware, using spoofed CERT-UA emails and password-protected ZIP archives to bypass email gateways. ### 5.
