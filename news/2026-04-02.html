🛡️ Briefing: April 02, 2026

### Daily Cyber Intelligence Brief Date: April 2, 2026 Reporting to: Technical Stakeholders / SOC Leadership Prepared by: Senior Cyber Security Researcher --- ### 1. Main Headline North Korean State Actor Sapphire Sleet Compromises Axios npm Package in Massive Supply Chain Attack --- ### 2. The Big Story On March 31, 2026, the popular JavaScript HTTP client Axios was the target of a successful supply chain compromise. The North Korean state-sponsored threat actor, tracked as Sapphire Sleet, managed to publish two malicious versions of the library to the npm registry: version 1.14.1 and 0.30.4. Given Axios’s role as a foundational dependency in the JavaScript ecosystem, the attack surface extends from individual developer environments to enterprise-scale production cloud infrastructures, potentially impacting millions of users. The technical mechanism involves a "poisoned" update that, once installed via standard package managers, triggers an automated download of second-stage payloads from a command-and-control (C2) server. Microsoft Threat Intelligence analysis confirms the malicious code was designed to establish persistence and facilitate further lateral movement. This attack coincides with a separate but related security incident involving Anthropic’s "Claude Code" tool, where a 59.8 MB source map leak led to the distribution of fake, malware-laden GitHub repositories. The threat cluster is leveraging these supply chain vulnerabilities to harvest sensitive environmental data. By targeting fundamental development tools (Axios) and emerging AI coding agents (Claude Code), the attackers are moving "upstream" to capture secrets, API keys, and SSH credentials at the source before they are even deployed to production. This represents a significant escalation in the industrialization of supply chain attacks targeting the DevOps pipeline. --- ### 3. Vendor Security Watch * Microsoft: Released guidance on mitigating the Axios npm compromise. Additionally, researchers detailed a new stealth tradecraft involving cookie-controlled PHP webshells in Linux environments that evade detection by hiding execution triggers within HTTP cookies and utilizing `php-fpm` for obfuscation. * Cisco: Issued a critical advisory for the Integrated Management Controller (IMC) and Software Manager (SSM). CVE-2026-20093 (CVSS 9.8) allows unauthenticated, remote attackers to bypass authentication and gain administrative access to the system. * Cloudflare: Introduced EmDash, a serverless JavaScript CMS that utilizes Worker isolates to sandbox plugins. This architecture aims to mitigate the "WordPress-style" plugin vulnerability surface by preventing cross-plugin data access. * Oracle: *No critical updates reported in this cycle.* * Fortinet: *No critical updates reported in this cycle.* --- ### 4. Critical Headlines * Chrome Zero-Day (CVE-2026-5281): Google has patched its fourth actively exploited zero-day of the year. The vulnerability resides in the WebGPU Dawn component and allows for remote code execution (RCE) in the browser context. * Next.js Exploit (CVE-2025-55182): A threat cluster tracked as UAT-10608 is exploiting the "React2Shell" vulnerability across 766 Next.js hosts. The operation uses the NEXUS Listener framework to harvest AWS secrets, GitHub tokens, and Stripe API keys. * $280M Drift Protocol Breach: Hackers successfully seized "Security Council" administrative powers within the Drift Protocol, allowing them to bypass governance controls and drain $280 million in assets. * CanisterWorm Wiper: A new worm is targeting cloud services in the Iranian region. The malware checks for Farsi language settings or Iranian time zones before initiating a data-wiping routine, signaling a shift toward localized, geopolitically motivated wiper attacks. * IoT Botnet Takedown: Federal authorities disrupted four major IoT botnets (Aisuru, Kimwolf, JackSkid, and Mossad) that controlled over three million hacked routers and cameras used for record-breaking DDoS attacks. --- ### 5.
