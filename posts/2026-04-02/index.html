DAILY CYBER INTELLIGENCE BRIEF April 2, 2026

1. Main Headline: Critical 9.8 CVSS Flaw in Cisco IMC Patched as UAT-10608 Exploits ‘React2Shell’ Vulnerability at Scale#

2. The Big Story: UAT-10608 Weaponizes CVE-2025-55182 for Automated Secret Exfiltration#

A massive credential harvesting operation, attributed to the threat cluster UAT-10608, has successfully breached at least 766 Next.js hosts by exploiting a vulnerability tracked as CVE-2025-55182 (colloquially known as “React2Shell”). The campaign serves as a high-efficiency vacuum for development and infrastructure secrets, specifically targeting database credentials, SSH private keys, AWS secrets, Stripe API keys, and GitHub tokens. The breadth of the exfiltration suggests a highly organized effort to facilitate secondary supply chain attacks or deep cloud environment penetration.

Technically, the attackers utilize an automated framework dubbed “NEXUS Listener.” Once initial access is achieved via the React2Shell vulnerability, the framework executes a series of scripts designed to parse environment variables and local configuration files. The operation specifically looks for shell command histories and .aws/credentials files, which often contain long-lived access keys that allow for persistence even after the initial web application vulnerability is patched.

The exploitation of Next.js environments highlights a shift toward targeting modern JavaScript frameworks that often sit at the intersection of front-end delivery and back-end cloud orchestration. By compromising these hosts, UAT-10608 effectively bypasses traditional perimeter defenses, gaining the same level of access as the legitimate developers or CI/CD pipelines associated with the affected applications.

3. Vendor Security Watch#

  • Cisco: Released critical patches for the Integrated Management Controller (IMC) and Software Manager (SSM) to address CVE-2026-20093 (CVSS 9.8). This flaw allows unauthenticated, remote attackers to bypass authentication and gain administrative privileges.
  • Microsoft: Issued an urgent mitigation advisory regarding a supply chain compromise of the Axios npm package. Attributed to the North Korean state actor Sapphire Sleet, the attack involved malicious updates designed to download second-stage payloads from a C2 server.
  • Microsoft: Disclosed technical analysis of stealthy PHP webshells in Linux environments that utilize HTTP cookies for command execution gating and php-fpm for obfuscated execution, evading standard detection signatures.
  • Cloudflare: Announced the beta of “EmDash,” a serverless CMS that attempts to mitigate plugin-based vulnerabilities (common in WordPress) by running all third-party code within sandboxed Worker isolates.
  • Cloudflare: Published results of an independent audit of the 1.1.1.1 public DNS resolver, confirming compliance with stated privacy and data-deletion policies.

4. Critical Headlines#

  • Drift Protocol Governance Breach: Attackers seized control of Security Council administrative powers in a sophisticated operation, resulting in the theft of $280 million.
  • Apple Issues Emergency iOS 18 Patch: A rare update for an older OS version was released to block the “DarkSword” exploit kit, which allows for silent device takeover via malicious websites.
  • CanisterWorm Wiper Attacks: A financially motivated group is deploying a wiper worm targeting cloud services; the malware specifically triggers data destruction on systems using the Farsi language or Iran’s time zone.
  • Global IoT Botnet Takedown: Federal authorities disrupted the infrastructure of four major botnets (Aisuru, Kimwolf, JackSkid, and Mossad) responsible for record-breaking DDoS attacks involving 3 million hacked devices.
  • Claude Code Impersonation: Threat actors are using fake GitHub repositories mimicking the leaked Claude Code source code to distribute the Vidar infostealer.

5. Admin Priority List#

  1. PATCH IMMEDIATELY: Apply updates to Cisco IMC and SSM to remediate CVE-2026-20093. This is a pre-auth remote compromise vector with a 9.8 CVSS.
  2. AUDIT NPM DEPENDENCIES: Verify all Axios library versions in JavaScript projects. Ensure no packages from the March 31, 2026, compromise window (attributed to Sapphire Sleet) remain in production or CI/CD pipelines.
  3. REMEDIATE NEXT.JS: Scan all public-facing Next.js instances for CVE-2025-55182. If a breach is suspected, rotate all AWS keys, GitHub tokens, and Stripe API secrets immediately.
