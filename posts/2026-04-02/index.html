1. Main Headline: Sapphire Sleet Compromises Axios Ecosystem in Massive NPM Supply Chain Attack#

2. The Big Story: North Korean State Actor Targets JavaScript Supply Chain#

On March 31, 2026, the North Korean state-sponsored threat actor Sapphire Sleet successfully executed a supply chain compromise against Axios, one of the most widely utilized HTTP clients in the JavaScript ecosystem. The attackers published two malicious npm packages disguised as legitimate version updates. These packages were engineered to automatically fetch secondary payloads from a command-and-control (C2) infrastructure managed by the threat actor. Although the malicious versions have been purged from the npm registry, the window of exposure potentially impacted millions of developers and automated CI/CD pipelines globally.

The compromise coincides with a surge in automated credential harvesting operations, specifically the activity of threat cluster UAT-10608. This group has been observed exploiting CVE-2025-55182 (also known as the React2Shell vulnerability) as an initial infection vector. By targeting Next.js hosts, the attackers are exfiltrating high-value secrets, including database credentials, SSH private keys, AWS secrets, Stripe API keys, and GitHub tokens. The technical synergy between supply chain poisoning and automated secret harvesting suggests a highly coordinated effort to gain long-term persistence in cloud environments.

The technical mechanism of the Axios compromise relies on the inherent trust in automated dependency managers. Once the malicious package is installed, it executes a post-install script or an obfuscated function within the library to initiate the C2 connection. This mirrors recent tradecraft observed in Linux environments, where “cookie-controlled” PHP webshells are being used for stealthy persistence. These shells hide execution triggers within HTTP cookies and leverage php-fpm and cron jobs to evade standard detection, allowing attackers to maintain access even after initial vulnerabilities are patched.

3. Vendor Security Watch#

  • Cisco: Released critical patches for the Integrated Management Controller (IMC) and Security Services Manager (SSM) addressing CVE-2026-20093 (CVSS 9.8). This vulnerability allows unauthenticated, remote attackers to bypass authentication and gain full system access with elevated privileges.
  • Microsoft: Mitigating the Sapphire Sleet supply chain attack on the Axios npm package; also issued technical guidance on detecting cookie-gated PHP webshells in Linux hosting environments that utilize php-fpm for execution.
  • Cloudflare: Completed an independent privacy audit of the 1.1.1.1 DNS resolver, confirming adherence to data-minimization policies; announced the beta of “EmDash,” a JavaScript-based CMS that sandboxes plugins in Worker isolates to mitigate the security risks associated with traditional CMS architectures like WordPress.
  • Apple: Issued an emergency backport patch for iOS 18 to address the “DarkSword” exploit. The urgency of the patch—targeting users who have not yet upgraded to iOS 26—indicates a significant risk of active exploitation by a sophisticated malware strain.

4. Critical Headlines#

  • Iran-Linked Wiper Attacks: A hacktivist group with ties to Iranian intelligence has claimed responsibility for a data-wiping attack on medical technology firm Stryker. Separately, the “CanisterWorm” malware is actively targeting cloud services, wiping data on systems with Farsi language settings or Iranian time zones.
  • Drift Protocol $280M Heist: The decentralized protocol suffered a catastrophic loss after threat actors compromised the “Security Council” administrative powers, gaining control over protocol governance to drain assets.
  • IoT Botnet Infrastructure Dismantled: Global law enforcement agencies (US, Canada, Germany) disrupted four major IoT botnets—Aisuru, Kimwolf, JackSkid, and Mossad—which controlled over 3 million compromised routers and cameras used for record-breaking DDoS attacks.
  • Vidar Malware via Claude Code Leak: Threat actors are utilizing fake GitHub repositories claiming to contain leaked “Claude Code” source code to distribute the Vidar infostealer.
  • Residential Proxy Reputation Bypass: New research indicates that 78% of 4 billion analyzed malicious sessions successfully evaded IP reputation checks by utilizing residential proxies, complicating the distinction between legitimate users and botnet traffic.

5. Admin Priority List#

  1. Immediate Patch: Deploy updates for Cisco IMC and SSM to remediate CVE-2026-20093. This is a CVSS 9.8 vulnerability and should be prioritized for all internet-facing management interfaces.
  2. Dependency Audit: Scan all JavaScript projects for Axios dependencies. Ensure builds do not include the malicious versions published on March 31–April 1, 2026. Rotate any AWS, GitHub, or Stripe keys that may have been exposed in environments using Next.js (targeting CVE-2025-55182).
  3. Emergency iOS Update: For organizations maintaining legacy devices, ensure the iOS 18 “DarkSword” security patch is applied immediately to prevent infection by the new malware strain.
