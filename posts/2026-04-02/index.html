DAILY CYBER INTELLIGENCE BRIEF

1. MAIN HEADLINE#

North Korean State Actor “Sapphire Sleet” Compromises Axios NPM Package in Massive Supply Chain Attack

2. THE BIG STORY#

Axios Dependency Poisoning and the Sapphire Sleet Campaign On March 31, 2026, the North Korean state-sponsored threat actor Sapphire Sleet successfully executed a supply chain compromise against Axios, one of the most widely used HTTP clients in the JavaScript ecosystem. The attackers published two malicious npm packages disguised as legitimate version updates. Once integrated into a project’s build pipeline, these packages were designed to communicate with a remote command-and-control (C2) server to download secondary payloads, effectively granting attackers a foothold within developer environments and production servers globally.

The scale of this compromise is significant, as Axios serves millions of users across nearly every sector. Organizations at risk include any enterprise utilizing modern JavaScript frameworks where npm install or automated CI/CD pipelines might have pulled the tainted versions. While the malicious versions have since been removed from the npm registry, the incident highlights a persistent vulnerability in the trust-based model of package management, where even highly-vetted libraries can be weaponized by sophisticated state actors.

Technically, the attack leveraged “dependency confusion” and account takeover tactics to inject the malicious code. Once executed, the scripts initiated an outbound connection to Sapphire Sleet’s infrastructure. This allowed for the exfiltration of environment variables, which often contain sensitive credentials, API keys, and AWS secrets. Microsoft Threat Intelligence confirms that this operation is part of a broader trend where North Korean actors shift from simple tool-based attacks to targeting the underlying cyberattack surface of the software supply chain itself.

3. VENDOR SECURITY WATCH#

  • Microsoft: Disclosed a supply chain compromise involving the Axios npm package attributed to Sapphire Sleet. Microsoft also issued warnings regarding cookie-controlled PHP webshells in Linux hosting environments, which use php-fpm and custom cookies to evade traditional detection.
  • Cisco: Released critical patches for the Integrated Management Controller (IMC) and Software Manager (SSM). The most severe flaw, CVE-2026-20093 (CVSS 9.8), allows unauthenticated remote attackers to bypass authentication and gain elevated system privileges.
  • Cloudflare: Introduced EmDash, a serverless CMS built on Astro 6.0, which attempts to mitigate plugin-based vulnerabilities by running them in sandboxed Worker isolates. They also reported that AI bot traffic now accounts for over 10 billion requests per week, necessitating a redesign of CDN caching logic.
  • Fortinet: No critical updates or CVEs reported in this data cycle.
  • Oracle: No critical updates or CVEs reported in this data cycle.

4. CRITICAL HEADLINES#

  • UAT-10608 Exploits Next.js (CVE-2025-55182): A threat cluster is leveraging the “React2Shell” vulnerability to breach Next.js hosts. The automated operation uses the “NEXUS Listener” framework to harvest SSH keys, GitHub tokens, and Stripe API keys from over 766 confirmed victims.
  • Apple Issues Emergency “DarkSword” Patch: Apple has taken the rare step of backporting security fixes to iOS 18 to combat the “DarkSword” exploit kit. The exploit is being used in active attacks, forcing updates even for users who have not migrated to iOS 26.
  • Drift Protocol $280M Governance Hack: Threat actors seized control of the Drift Protocol’s “Security Council” administrative powers. This sophisticated operation allowed the attackers to drain $280 million in assets by manipulating protocol-level permissions.
  • CanisterWorm Targeting Iran: A financially motivated group has deployed “CanisterWorm,” a wiper malware spreading through cloud services. The malware specifically targets systems using the Farsi language or Iran’s time zone, wiping data upon infection.
  • Global IoT Botnet Takedown: The U.S. DOJ and international partners dismantled the infrastructure for four major botnets—Aisuru, Kimwolf, JackSkid, and Mossad. These botnets controlled over 3 million compromised IoT devices used for record-breaking DDoS attacks.

5. ADMIN PRIORITY LIST#

  1. Immediate Dependency Audit: Scan all JavaScript projects for Axios versions published on or around March 31, 2026. Revert to known-good versions and rotate any environment secrets (API keys, DB credentials) present in those build environments.
  2. Patch Cisco IMC/SSM: Apply updates for CVE-2026-20093 immediately. This is a 9.8 CVSS unauthenticated bypass and is a prime target for initial access brokers.
  3. Remediate Next.js Environments: Ensure all Next.js deployments are patched against CVE-2025-55182. Audit systems for the presence of the “NEXUS Listener” or unauthorized outgoing connections to prevent credential harvesting by UAT-10608.
