Daily Cyber Intelligence Brief#

Main Headline: Russian APT28 Hijacks 18,000+ SOHO Routers to Siphon Microsoft Office Auth Tokens#

The Big Story The Russian-linked threat actor APT28 (also known as Forest Blizzard) has launched a massive global DNS hijacking and credential theft campaign targeting insecure MikroTik and TP-Link SOHO routers. Since at least May 2025, the group has been exploiting known vulnerabilities in older router firmware to modify device settings, effectively turning consumer-grade hardware into malicious infrastructure for state-sponsored espionage.

The technical mechanism of this campaign is particularly stealthy; rather than deploying traditional malware on end-user machines, APT28 harvests Microsoft Office authentication tokens directly from the network traffic of compromised routers. By intercepting these tokens, the actors can bypass Multi-Factor Authentication (MFA) and gain persistent access to enterprise cloud environments. Reports indicate that over 18,000 networks have been impacted, with the attackers focusing on siphoning data without the need for a persistent file-based footprint on the target’s workstation.

This campaign highlights a critical shift toward “living off the land” at the network edge. By weaponizing the transition of data between the SOHO environment and SaaS providers, APT28 exploits the visibility gap inherent in unmanaged edge devices. Organizations with remote workers are at the highest risk, as these “zombie vulnerabilities” in unpatched home routers now serve as a direct pipeline into hardened corporate Microsoft 365 tenants.

Vendor Security Watch

  • Cloudflare: Launched “Organizations” in public beta, introducing a consolidated authorization management layer to handle multiple enterprise accounts and identity systems at scale.
  • Cisco: Talos reports a surge in “React2Shell” attacks and the weaponization of SaaS notification pipelines (e.g., Slack, Teams) to deliver phishing payloads.
  • Microsoft: Impacted by APT28 token harvesting; administrators are advised to audit sign-in logs for unusual IP addresses associated with SOHO ranges and enforce session lifetime limits.
  • Fortinet / Oracle: No critical security updates or vulnerability disclosures were identified in today’s raw data feed.

Critical Headlines

  • Docker Engine AuthZ Bypass: A high-severity vulnerability, CVE-2026-34040 (CVSS 8.8), has been identified. The flaw is an incomplete fix for CVE-2024-41110 and allows attackers to bypass authorization plugins to gain host access.
  • Ninja Forms RCE: A critical vulnerability in the Ninja Forms File Uploads premium add-on for WordPress is being actively exploited. It allows unauthenticated arbitrary file uploads, leading to full remote code execution (RCE).
  • Snowflake Data Theft: A third-party SaaS integrator breach has led to the theft of authentication tokens, resulting in data exfiltration for over a dozen Snowflake customers.
  • CanisterWorm Wiper: A new worm is targeting Iranian infrastructure, specifically seeking out systems with Farsi language settings to execute data-wiping routines.
  • FBI Crime Report: The FBI confirms record losses of $21 billion to cybercrime in the last year, with Business Email Compromise (BEC) and investment scams remains the primary drivers.

Admin Priority List

  1. Patch Docker: Immediately update Docker Engine to resolve CVE-2026-34040 to prevent unauthorized host-level access via plugin bypass.
  2. Edge Device Audit: Force firmware updates on all MikroTik and TP-Link routers; change default credentials and disable remote management interfaces to mitigate APT28 activity.
  3. WordPress Remediation: Update the Ninja Forms File Uploads extension to the latest version or disable the plugin until a verified patch is applied to prevent unauthenticated RCE.
