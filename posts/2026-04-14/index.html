Daily Cyber Intelligence Brief#

Russian Military Intelligence Weaponizes Legacy Routers to Harvest Microsoft Office Tokens#

The Big Story Russian military intelligence units (GRU) have launched a massive campaign targeting older internet routers to harvest authentication tokens from Microsoft Office users. The operation has successfully compromised more than 18,000 networks globally, allowing state-backed actors to quietly siphon session data without the need to deploy traditional malware or persistent backdoors. By targeting the edge infrastructure rather than the endpoint, the actors effectively bypass many host-based security controls and initial perimeter defenses.

The technical mechanism involves exploiting known vulnerabilities in legacy router firmware to gain a foothold. Once the router is compromised, the attackers intercept or manipulate traffic to capture authentication tokens—specifically those used for Microsoft Office 365 services. These tokens, which are generated after a user successfully authenticates (often via MFA), can be replayed by the attackers to gain unauthorized access to cloud-based email and document repositories, effectively session-jacking the user’s identity.

This campaign underscores the “collapse of the patch window” for edge devices. Because these routers often lack automated update mechanisms or are “set and forget” hardware, they remain vulnerable to N-day exploits long after patches are available. The GRU’s ability to conduct this at a scale of 18,000+ networks suggests a high degree of automation in their scanning and exploitation framework, turning common SOHO and enterprise edge hardware into a distributed proxy network for credential theft.

Vendor Security Watch#

  • Cloudflare: Announced the General Availability (GA) of Sandboxes, providing isolated, persistent environments for AI agents. Additionally, released a new unified CLI (“cf”) to manage approximately 3,000 API operations and “Durable Object Facets” for isolating SQLite databases in dynamic worker environments.
  • Fortinet: A critical SQL injection vulnerability in FortiClient EMS (CVE-2026-21643, CVSS 9.1) has been added to the CISA Known Exploited Vulnerabilities (KEV) catalog. The flaw allows unauthenticated attackers to execute arbitrary code via specially crafted requests.
  • Microsoft: Detailed the activity of Storm-2755, a financially motivated actor targeting Canadian organizations through payroll redirection. Also issued a warning regarding an intent-redirection vulnerability in a common third-party Android SDK that exposes sensitive data in cryptocurrency wallet applications.
  • Cisco: Talos researchers released a technical analysis on the “collapse of the patch window,” noting that AI-assisted exploitation is significantly reducing the time administrators have to remediate vulnerabilities before active exploitation begins.
  • Oracle: No major security advisories or critical patch updates were reported in today’s primary intelligence feeds.

Critical Headlines#

  • ShowDoc RCE Under Active Exploitation: The document management platform ShowDoc is facing active exploitation of CVE-2025-0520 (CVSS 9.4). The flaw involves unrestricted file uploads leading to Remote Code Execution on unpatched servers.
  • wolfSSL Library Forged Certificate Risk: A critical flaw in the wolfSSL library allows for improper verification of hash algorithms during ECDSA signature checks, potentially enabling the use of forged SSL/TLS certificates.
  • Polkadot Bridge Governance Exploit: An attacker gained administrative control over a Polkadot cross-chain bridge, allowing them to mint and dump 1 billion DOT tokens, significantly impacting the DeFi ecosystem’s liquidity.
  • CanisterWorm Wiper Attacks: A new worm, dubbed “CanisterWorm,” is targeting Iranian infrastructure. The malware spreads via misconfigured cloud services and executes a destructive wiper payload if the system’s time zone or language is set to Farsi.
  • REvil Leader Doxed: German authorities have identified and doxed the individual “UNKN,” the alleged head of the REvil and GandCrab ransomware operations, as 31-year-old Russian national Daniil Maksimovich Shchukin.

Admin Priority List#

  1. Remediate FortiClient EMS: Immediately patch FortiClient EMS to address CVE-2026-21643; this is now a CISA-confirmed exploited vulnerability.
  2. Audit Edge Device Firmware: Conduct a comprehensive audit of all legacy routers and edge devices. Disable unused services and update firmware to mitigate the GRU-led token harvesting campaign.
  3. Patch ShowDoc Instances: Organizations using ShowDoc must update to the latest version immediately to block active exploitation of the CVE-2025-0520 RCE flaw.
