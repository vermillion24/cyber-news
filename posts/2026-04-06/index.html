Daily Cyber Intelligence Brief Date: April 4, 2026 Analyst: Senior Security Researcher

1. Main Headline#

Active Exploitation of Critical FortiClient EMS Vulnerability (CVE-2026-35616) Prompts Emergency Out-of-Band Patch

2. The Big Story#

Critical Pre-Auth API Bypass in FortiClient EMS Under Wild Exploitation Fortinet has issued an emergency security update to address a critical vulnerability in its FortiClient Enterprise Management Server (EMS), tracked as CVE-2026-35616 (CVSS score: 9.1). The flaw is characterized as an improper access control vulnerability [CWE-284] within the management server’s API. Because the flaw allows for a pre-authentication bypass, remote unauthenticated attackers can gain unauthorized access to administrative functions without providing valid credentials.

The vulnerability is particularly dangerous because the EMS serves as the centralized hub for managing security postures across thousands of corporate endpoints. Exploitation allows an attacker to escalate privileges and potentially achieve full administrative control over the EMS. Once the server is compromised, the attacker can manipulate security policies, deploy malicious configurations, or execute code on any endpoint connected to the management fabric, facilitating massive internal lateral movement.

Telemetry indicates that threat actors are already actively exploiting this flaw in targeted attacks. While Fortinet has not publicly named the specific actors involved, the mechanism of the exploit—targeting edge management infrastructure—aligns with recent patterns of state-sponsored activity aimed at gaining persistent access to enterprise environments. Organizations using FortiClient EMS must prioritize this update, as the exploit effectively nullifies the perimeter security provided by the management suite.

3. Vendor Security Watch#

  • Fortinet: Released emergency patches for CVE-2026-35616 impacting FortiClient EMS. The update addresses a pre-authentication API bypass that enables privilege escalation.
  • Microsoft: Issued technical analysis and mitigation guidance for the Axios npm supply chain compromise. Microsoft also released research on a new “cookie-controlled” PHP webshell tradecraft used in Linux environments, which utilizes obfuscated php-fpm execution and HTTP cookie-gated triggers for stealthy persistence.
  • Cisco (Talos): Published a detailed forensic breakdown of the Axios NPM incident and warned of the “democratization” of Business Email Compromise (BEC) tools, which now allow lower-skilled actors to execute sophisticated financial fraud.
  • Cloudflare: Announced a structural redesign of CDN caching mechanisms to handle the 10-billion-request-per-week surge in AI bot traffic. Separately, an independent audit of the 1.1.1.1 public DNS resolver confirmed the efficacy of its privacy-preserving architecture.
  • Oracle: No critical updates or vulnerability disclosures reported in today’s data set.

4. Critical Headlines#

  • $285M Drift Protocol Hack Attributed to DPRK: Forensic analysis of the April 1st theft from the Solana-based exchange traces the attack to a six-month social engineering campaign by North Korean state actors.
  • North Korean “Sapphire Sleet” Hijacks Axios Library: The popular NPM package axios was compromised to deliver malicious payloads from actor-controlled C2 infrastructure; millions of JavaScript applications were potentially exposed during the version hijacking.
  • “CanisterWorm” Wiper Targeting Iran: A new malware strain is spreading via misconfigured cloud services, specifically targeting systems set to the Iran time zone or Farsi language with data-wiping payloads.
  • Dismantling of IoT Botnets: International law enforcement disrupted the infrastructure of four major botnets—Aisuru, Kimwolf, JackSkid, and Mossad—responsible for record-breaking DDoS attacks involving over three million infected devices.
  • Next.js Applications Exploited via React2Shell: Hackers are conducting automated credential-harvesting campaigns by exploiting CVE-2025-55182 in vulnerable Next.js framework deployments.
  • German Authorities Dox REvil Leader: Investigators have identified the head of the REvil and GandCrab ransomware gangs (“UNKN”) as 31-year-old Russian national Daniil Maksimovich Shchukin.

5. Admin Priority List#

  1. PATCH IMMEDIATELY: Update FortiClient EMS to the latest version to mitigate CVE-2026-35616. This is a pre-auth exploit with confirmed activity in the wild.
  2. DEPENDENCY AUDIT: Run npm audit and manually verify Axios library versions. Ensure all internal projects have rolled back from the compromised Sapphire Sleet versions to a verified clean release.
  3. VULNERABILITY SCAN: Identify and patch any Next.js instances vulnerable to CVE-2025-55182 to prevent automated credential theft and shell access.
