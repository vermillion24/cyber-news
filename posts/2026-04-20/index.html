April Patch Tuesday: Microsoft Fixes 167 Flaws, Including SharePoint Zero-Day and “BlueHammer” Bypass#

The April 2026 Patch Tuesday cycle has arrived with a massive release, addressing 167 security vulnerabilities across Microsoft’s software stack. Most critical in this release are a zero-day vulnerability in SharePoint Server and a publicly disclosed weakness in Windows Defender, tracked as “BlueHammer.” The update coincides with emergency patches from Adobe for an actively exploited RCE in Acrobat/Reader and Google’s fourth zero-day fix for Chrome this year, signaling a high-intensity month for enterprise remediation teams.

The SharePoint Server zero-day allows for unauthenticated remote code execution (RCE) in specific configurations, posing a severe threat to corporate intranets. Meanwhile, the “BlueHammer” vulnerability in Windows Defender represents a significant breakdown in the operating system’s native defense-in-depth. By exploiting this flaw, threat actors can bypass endpoint protection scans or suppress alerts, allowing malicious payloads to execute without triggering the standard behavioral blocks that organizations rely on for automated threat containment.

Technically, these vulnerabilities highlight a dangerous trend in legacy debt and the erosion of “trusted” perimeters. The SharePoint flaw exploits improper handling of server-side data, while BlueHammer targets the trust relationship between the OS kernel and its security modules. Organizations are at extreme risk of initial entry via SharePoint and subsequent lateral movement that remains invisible to host-based defenses. The sheer volume of 167 CVEs suggests that attackers have a wide range of secondary vectors to exploit if primary patches are not applied in a tiered, prioritized manner.

Vendor Security Watch#

  • Microsoft: Released updates for 167 CVEs, including a critical SharePoint zero-day and the “BlueHammer” Windows Defender bypass. Patches also address credential exposure risks in Dynamics 365 and Power Platform through new platform engineering controls.
  • Cisco: Talos disclosed patches for one vulnerability in Foxit Reader and six vulnerabilities in the LibRaw file reader. Additionally, Cisco identified the “PowMix” botnet using undocumented techniques to target Czech infrastructure.
  • Cloudflare: Announced “Agents Week” security features, including new AI Gateway routing controls that have already processed 20 million internal requests to mitigate agentic AI risks.
  • Fortinet: No critical updates reported in this data cycle.
  • Oracle: No critical updates reported in this data cycle.

Critical Headlines#

  • SGLang Critical RCE (CVE-2026-5760): A CVSS 9.8 vulnerability in the SGLang AI serving framework allows for remote code execution via malicious GGUF model files. The flaw is a command injection vulnerability that triggers when processing untrusted model metadata.
  • GRU Router Campaign: Russian military intelligence units are exploiting known flaws in legacy routers to harvest Microsoft Office authentication tokens from over 18,000 networks. The attack requires no malware on the endpoint, instead siphoning tokens directly from network traffic.
  • Kelp DAO $291M Exploit: An infrastructure breach linked to Kelp DAO has resulted in a nearly $300 million loss, triggering a $6.2 billion liquidity crunch on the Aave DeFi platform as users panic-withdraw assets.
  • REvil/GandCrab Leader Identified: German authorities have unmasked “UNKN,” the alleged head of the REvil ransomware gang, as 31-year-old Daniil Maksimovich Shchukin, following a multi-year investigation into 130 acts of computer sabotage.
  • Vercel Data Breach: Cloud development giant Vercel has confirmed a security incident after threat actors began selling data allegedly exfiltrated from its internal systems.

Admin Priority List#

  1. Immediate Patching: Prioritize the Microsoft April 2026 security updates, specifically for SharePoint Server and Windows Defender (BlueHammer).
  2. AI Stack Audit: Scan for the use of SGLang in production environments and block the loading of GGUF files from untrusted sources to mitigate CVE-2026-5760.
  3. Token Rotation: Organizations using legacy edge routers should audit for GRU-linked activity, rotate all Microsoft 365 authentication tokens, and implement hardware-backed MFA to counter token-theft techniques.
