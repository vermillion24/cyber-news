Daily Cyber Intelligence Brief#

Main Headline: Russian Military Hijacks 18,000 SOHO Routers to Siphon Microsoft Office Tokens

The Big Story Microsoft and security researchers have identified a massive campaign orchestrated by “Forest Blizzard” (APT28), a threat group linked to Russian military intelligence (GRU). The operation has successfully compromised over 18,000 networks by targeting insecure Small Office/Home Office (SOHO) equipment, specifically older EdgeRouter and Ubiquiti devices. By repurposing these routers as malicious infrastructure, the threat actor creates a stealthy pivot point to launch attacks against corporate and government targets while appearing as legitimate home-office traffic.

The primary objective of this campaign is the mass harvesting of Microsoft Office authentication tokens. Unlike traditional attacks that require infecting a victim’s workstation with malware, this method allows the threat actors to gain persistent access to cloud-based productivity suites while remaining entirely “fileless” on the target’s host machine. By capturing these session tokens, the attackers effectively bypass Multi-Factor Authentication (MFA), as the stolen tokens grant them the same authenticated status as the legitimate user.

Technically, the attack is executed through a combination of DNS hijacking and Adversary-in-the-Middle (AitM) techniques implemented at the router level. Once a router is compromised—often via known vulnerabilities or weak credentials—the actor modifies the device’s configuration to redirect authentication traffic to attacker-controlled infrastructure. When a user attempts to log into Microsoft services, the credentials and session tokens are intercepted in transit. This strategy leverages the lack of visibility most organizations have into their employees’ home networking hardware.

Vendor Security Watch

  • Cloudflare: Launched “Organizations” into public beta. This new management layer consolidates authorization systems to enable centralized, org-wide account management for enterprise customers with multiple accounts. Cloudflare also moved its target for full post-quantum security to 2029 due to accelerated quantum hardware development.
  • Fortinet: No critical vulnerability updates or security patches were reported in today’s intelligence cycle.
  • Oracle: No critical security updates or vulnerability patches were reported in today’s intelligence cycle.
  • Cisco: Cisco Talos identified “LucidRook,” a new Lua-based malware loader used in spear-phishing attacks against Taiwanese NGOs. They also warned of a rise in attackers weaponizing SaaS notification pipelines (e.g., collaboration platform alerts) to bypass email filters.
  • Microsoft: Deployed a server-side fix for a Windows 11 23H2 bug that broke the Start Menu search feature. Microsoft also published findings on “Storm-1175,” a high-tempo Medusa ransomware affiliate that rapidly weaponizes web-facing vulnerabilities for data exfiltration.

Critical Headlines

  • Ninja Forms RCE: A critical vulnerability in the Ninja Forms File Uploads premium add-on for WordPress allows unauthenticated arbitrary file uploads, which can be leveraged for remote code execution (RCE).
  • Supply Chain Poisoning: North Korean threat actors (“Contagious Interview”) have flooded the npm, PyPI, Go, and Rust ecosystems with 1,700 malicious packages designed to impersonate legitimate developer tools and load malware.
  • Industrial Infrastructure Targeted: U.S. agencies have issued warnings regarding Iran-linked actors targeting internet-exposed Programmable Logic Controllers (PLCs) across critical infrastructure sectors, leading to operational disruptions.
  • CanisterWorm Wiper: A new financially motivated group has unleashed “CanisterWorm,” a wiper that spreads through poorly secured cloud services and targets systems with Farsi language settings or Iranian time zones.

Admin Priority List

  1. Harden Edge Devices: Audit all SOHO/EdgeRouter hardware. Disable remote management interfaces, update to the latest firmware, and enforce complex credentials to mitigate Forest Blizzard token-theft tactics.
  2. Patch WordPress Add-ons: Immediately update the Ninja Forms File Uploads add-on to the latest version to prevent unauthenticated server compromise.
  3. Sanitize Development Pipelines: Implement automated scanning for all external dependencies in Go, Rust, Python, and Node.js projects to identify and remove malicious “Contagious Interview” packages.
