DAILY CYBER INTELLIGENCE BRIEF DATE: April 4, 2026 PRODUCED BY: Senior Security Research Office

1. MAIN HEADLINE#

Fortinet Issues Emergency Patch for CVE-2026-35616 Amid Active Exploitation of FortiClient EMS

2. THE BIG STORY: North Korean Actors Execute Multi-Stage Supply Chain Attack via Axios NPM Ecosystem#

The popular JavaScript HTTP client Axios has been confirmed as the primary target of a sophisticated supply chain compromise attributed to the North Korean state-sponsored group Sapphire Sleet. The attack originated with a highly targeted social engineering campaign against a maintainer, who was lured into running a malicious “fix” for a simulated Microsoft Teams error. This allowed the threat actor to hijack the maintainer’s credentials and publish two malicious versions of the Axios package to the NPM registry.

The technical mechanism involved the delivery of second-stage payloads from actor-controlled Command and Control (C2) infrastructure once the poisoned packages were integrated into downstream projects. Microsoft Threat Intelligence confirms that the malicious versions utilized postinstall.js scripts to execute arbitrary code upon installation. Because Axios is a foundational dependency for millions of web applications, the reach of this compromise is potentially vast, moving beyond simple credential theft to persistent environment access.

This incident highlights a broader trend of “democratized” supply chain attacks. Simultaneously, researchers discovered 36 additional malicious NPM packages disguised as Strapi CMS plugins. These packages were designed to exploit Redis and PostgreSQL instances, deploying reverse shells and persistent implants. The convergence of social engineering with automated dependency poisoning suggests a tactical shift toward high-leverage account takeovers of trusted open-source maintainers rather than simple typosquatting.

3. VENDOR SECURITY WATCH#

  • Fortinet: Released out-of-band patches for CVE-2026-35616 (CVSS 9.1). This is a critical improper access control vulnerability in FortiClient EMS that allows pre-authentication API access bypass, leading to privilege escalation. It is currently being exploited in the wild.
  • Microsoft: Issued technical guidance on a new “Cookie-controlled” PHP webshell tradecraft targeting Linux hosting environments. The shells use obfuscated code and php-fpm execution to hide behind specially crafted HTTP cookies, evading traditional signature-based detection.
  • Cloudflare: Introduced EmDash, a serverless CMS designed to mitigate the security risks inherent in WordPress-style plugin architectures. EmDash executes plugins within sandboxed Worker isolates, providing hard multi-tenancy and preventing lateral movement from compromised third-party code.
  • Cisco (Talos): Released a post-mortem on the Axios NPM incident, detailing the shift in supply chain tactics. Talos also reported on the “democratization” of Business Email Compromise (BEC), noting a surge in specialized kits that lower the barrier to entry for complex fraud.
  • Oracle: No major vulnerability disclosures or technical security updates were recorded in the provided telemetry for this period.

4. CRITICAL HEADLINES#

  • Wiper Attacks Surge in Middle East: A new worm dubbed ‘CanisterWorm’ is targeting cloud services in Iran, specifically wiping systems that utilize Farsi language settings or Iranian time zones. Parallel reports indicate an Iran-backed group has claimed a wiper attack against medical technology firm Stryker.
  • OAuth 2.0 Abuse: Device code phishing attacks have surged 37x this year. These attacks abuse the Device Authorization Grant flow to bypass MFA and hijack user sessions without requiring a traditional phishing landing page.
  • LinkedIn “BrowserGate”: Research reveals LinkedIn is utilizing hidden JavaScript to scan visitors’ browsers for over 6,000 specific Chrome extensions, raising significant privacy and reconnaissance concerns regarding the collection of local device data.
  • IoT Botnet Takedown: Federal authorities disrupted infrastructure for four major botnets—Aisuru, Kimwolf, JackSkid, and Mossad—which controlled over three million compromised IoT devices used for record-breaking DDoS attacks.
  • TA416 Targets Europe: The China-linked actor TA416 (RedDelta) has resumed targeting European diplomatic entities using PlugX malware and OAuth-based phishing lures.

5. ADMIN PRIORITY LIST#

  1. PATCH IMMEDIATELY: Update FortiClient EMS to the latest version to remediate CVE-2026-35616. This is an actively exploited pre-auth vulnerability.
  2. DEPENDENCY AUDIT: Run npm audit or use an SCA tool to ensure no versions of Axios or Strapi-related plugins from the March 31–April 2 window remain in your CI/CD pipelines.
  3. HARDEN OAUTH FLOWS: Review and potentially disable the OAuth 2.0 Device Authorization Grant if it is not required for your environment to mitigate the 37x surge in device code phishing.
