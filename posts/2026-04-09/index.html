Daily Cyber Intelligence Brief#

Main Headline: Russian Military Intelligence Weaponizes 18,000 SOHO Routers for Mass Token Theft

The Big Story The Russian state-sponsored threat actor Forest Blizzard (APT28), linked to the GRU, has compromised over 18,000 small-office and home-office (SOHO) routers globally to facilitate large-scale authentication token harvesting. By exploiting known vulnerabilities in legacy and unpatched edge devices, the actors have successfully turned consumer-grade hardware into a distributed malicious infrastructure designed to intercept enterprise-level communications.

This campaign places any organization with remote workers or unmanaged edge devices at extreme risk. Unlike traditional malware attacks that reside on the endpoint, this operation targets the network layer to bypass host-based security controls. By siphoning Microsoft Office authentication tokens directly from the network stream, the attackers can gain unauthorized access to corporate environments without triggering traditional malware alerts or requiring the victim to execute malicious files.

The technical mechanism involves DNS hijacking and Adversary-in-the-Middle (AiTM) tactics. Once the SOHO router is compromised, Forest Blizzard modifies the device’s DNS settings or routing tables to intercept traffic destined for legitimate Microsoft authentication endpoints. This allows the actor to capture session tokens in real-time. Furthermore, recent reports indicate the deployment of a new malware suite dubbed PRISMEX, which utilizes advanced steganography and COM hijacking to maintain persistence and facilitate command-and-control (C2) via legitimate cloud services.

  • Microsoft: Issued technical breakdowns of Storm-1175 (Medusa ransomware) operations targeting web-facing assets. Microsoft also warned of AI-driven device code phishing campaigns that automate the generation of live authentication codes to scale account compromises.
  • Fortinet: No new critical vulnerability disclosures or patches were reported in today’s cycle.
  • Oracle: No new critical vulnerability disclosures or patches were reported in today’s cycle.

Critical Headlines

  • UNC6783 Targets Zendesk: A new threat actor is compromising Business Process Outsourcing (BPO) providers to steal corporate Zendesk support tickets, gaining access to sensitive customer data.
  • Chaos Botnet Evolves: The Chaos malware variant has expanded its target set from edge devices to misconfigured cloud deployments, adding SOCKS proxy capabilities for stealthier lateral movement.
  • CanisterWorm Wiper Attacks: A financially motivated group has pivoted to geopolitical sabotage, deploying “CanisterWorm”—a wiper that targets systems with Farsi language settings or Iranian time zones.
  • Anthropic “Mythos” Bug Hunter: Anthropic’s Project Glasswing has revealed an AI model capable of autonomously chaining exploits and identifying vulnerabilities at a scale that exceeds human manual analysis.
  • Magento SVG Stealers: Threat actors are hiding credit card skimming code within pixel-sized Scalable Vector Graphics (SVG) files on Magento e-commerce platforms to evade signature-based detection.

Admin Priority List

  1. Harden Edge Infrastructure: Audit all SOHO and remote-office routers; immediately replace end-of-life hardware and apply firmware patches to mitigate APT28 DNS hijacking.
  2. Verify Web-Facing Assets: Conduct immediate vulnerability scans on all public-facing servers to identify flaws currently being weaponized by Storm-1175 for Medusa ransomware deployment.
  3. Monitor SaaS Auth Logs: Review Microsoft Entra ID and Zendesk logs for anomalous token usage or logins from BPO-associated IP ranges to detect AiTM or ticket theft.
