Main Headline: North Korean "Sapphire Sleet" Compromises Axios NPM Packages in Massive Supply Chain Attack ### 2. The Big Story: Axios Dependency Poisoning On March 31, 2026, the North Korean state-sponsored threat actor Sapphire Sleet successfully compromised the supply chain for Axios, one of the most widely used JavaScript HTTP clients. The attackers published two malicious versions to the npm registry: v1.14.1 and v0.30.4. These packages were configured to initiate a multi-stage infection chain that downloads payloads from a remote Command and Control (C2) server. The threat covers a massive surface area; Axios is a foundational dependency for millions of web applications and microservices. Any automated CI/CD pipeline or developer environment that pulled the latest minor or patch version during the exposure window is compromised. While the malicious packages have been removed from npm, the focus now shifts to incident response for organizations that may have already integrated the tainted code into their production environments. Technically, the malicious versions functioned as a downloader. Upon installation or execution, the scripts established a connection to Sapphire Sleet's infrastructure. This mechanism is consistent with North Korean "Lazarus-linked" tactics aimed at infiltrating corporate networks to facilitate either financial theft or intellectual property espionage. Organizations are advised to audit their `package-lock.json` files and build artifacts for the presence of the affected versions immediately. ### 3.
